# India Payment Pulse

## Project Overview
A comprehensive analysis and visualization of payment trends in India.

## Folder Structure
- `data/` - Data storage (raw, clean, reports)
- `scripts/` - Python scripts for data processing
- `sql/` - SQL queries and database scripts
- `logs/` - Application logs
- `dashboard/` - Dashboard and visualization files

## Getting Started
1. Install dependencies: `pip install -r requirements.txt`
2. Run data generation: `python scripts/generate_data.py`

## Requirements
See `requirements.txt` for Python dependencies.
